I built this website as a place to share with the world many of the things I have created.

I built the website using Google App Engine, webapp2, Python, Jinja2, jQuery, Bootstrap, and of course HTML, CSS, and JavaScript. Using Jinja2 for templating will make it easy to change the style of the website at some point in the future (it's very simple right now).

I use Google Analytics on this website. To opt out, install a Do Not Track Plugin for Google Chrome or enable the Do Not Track feature in Firefox.

I am open to suggestions on how to improve this website from a technical perspective.
